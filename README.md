## Featured Project

### 🎼 [Air Conductor](YOUR_REPO_URL)

Real-time gesture-driven music conductor that runs entirely in the browser. Detects hand landmarks via MediaPipe Hands, maps gesture dynamics to tempo and articulation, and synthesizes audio through the Web Audio API and Tone.js. Includes a custom peak-and-drop beat detection algorithm and an Elder Mode for accessibility.

> IEEE paper submitted · May 2025
